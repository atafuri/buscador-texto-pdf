🔍 Buscador de Texto em PDF
Do Caos ao Fluxo

Automação criada a partir de uma necessidade real:
buscar rapidamente informações dentro de múltiplos arquivos PDF de forma prática e escalável.

🚀 Sobre o projeto

Este programa permite localizar palavras-chave dentro de vários arquivos PDF de uma pasta selecionada, gerando um relatório estruturado com os resultados.

👉 Ideal para:

Engenheiros
Analistas
Profissionais administrativos
Qualquer pessoa que lide com grande volume de documentos

⚙️ Funcionalidades

✔ Busca por múltiplas palavras-chave
✔ Processamento em lote de PDFs
✔ Interface simples e intuitiva
✔ Geração de relatório em Excel
✔ Automação rápida e eficiente

📦 Como usar
Execute o arquivo .exe
Insira as palavras-chave (separadas por ;)
Selecione a pasta com os PDFs
Escolha onde salvar o resultado
Clique em Iniciar
